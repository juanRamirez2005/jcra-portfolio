## Instrucciones:

1. Agrega tu foto profesional como `profile.jpg`
2. Crea las diferentes versiones de iconos usando herramientas como:
   - https://realfavicongenerator.net/
   - https://www.favicon-generator.org/
3. Optimiza todas las imágenes para web (WebP recomendado para mejor rendimiento)
4. Mantén los nombres de archivo exactos para que las referencias funcionen correctamente

## Optimización SEO:
- Usa alt text descriptivo en todas las imágenes
- Optimiza el tamaño de archivo sin perder calidad
- Considera usar formato WebP para mejor compresión
- Asegúrate de que las imágenes tengan las dimensiones correctas para cada uso
